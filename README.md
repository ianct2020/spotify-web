# spotify-tools

Web app personal para administrar mi biblioteca de Spotify.

## Features

- **Sync Mirror** — Playlist espejo exacto de Liked Songs
- **Dedupe** — Encontrar y limpiar duplicados por playlist
- **Huérfanas** — Likes que no están en ninguna playlist
- **Zombis** — Tracks eliminados del catálogo
- **Versiones** — Mismo tema en distintos álbumes (remaster, live, etc.)

## Stack

HTML + CSS + JS vanilla, OAuth PKCE, GitHub Pages.
